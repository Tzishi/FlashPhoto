Detailed Description

Base class for filters that use convolution.

A Convolution Filter describes an algorithm for producing a filtered pixel based on a PixelBuffer and a set of coordinates in that PixelBuffer. It makes use of a kernel to determine the amounts that neighboring pixels affect the output color.
